# sshmngr

`sshmngr` is a lightweight SSH connection manager with interactive fuzzy search,
jumphost support, and secure credential storage via the OS keyring.

## Quick start
```bash
git clone https://github.com/YOURUSER/sshmngr.git
cd sshmngr
pip install .
```
Run it:
```bash
sshmngr
sshmngr webserver1
```
